## Ionic 1

<hr>


### 1. Install Ionic
```
npm i -g @ionic/cli
```

&nbsp;

### 2. What is Ionic

> Ionic is an open source UI toolkit for building performant, high-quality mobile apps using web technologies — HTML, CSS, and JavaScript — with integrations for popular frameworks like Angular, React, and Vue

&nbsp;

### 3. Start an App

```
ionic start
```

&nbsp;

### 4. Run the App

```
$ cd myApp
$ ionic serve
```
