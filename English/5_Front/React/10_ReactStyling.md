### 1. Using plain CSS

Create React App has already enabled the use of plain CSS in the project. In fact, if you look in App.tsx, it already uses plain CSS:
- App.tsx
    ```typescript
    import './App.css';

    function App() {
        return (
            <div className="App">
                ...
            </div>
        );
    }
    ```
